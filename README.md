# Weather-Application

Project Overview
Weather Application is a simple and interactive weather app designed to let users check current weather conditions for any city or location. It’s built to fetch real-time weather data from a weather API and display important weather metrics in a clean, easy-to-read interface.
This app is ideal for anyone who wants a straightforward way to check temperature, humidity, wind speed, and other basic weather details using an intuitive UI.

Features
✔️ Search weather by city name
✔️ Display current temperature, humidity & wind speed
✔️ Simple and responsive design
✔️ Fetches live weather data through an API

Technologies Used
Frontend: HTML, CSS, JavaScript (or Java/Kotlin if it’s Android — replace accordingly)
API Used: OpenWeatherMap API (or whichever API your project integrates)

How It Works
User enters a city name in the search field.
The application sends a request to the weather API.
Weather data is received and displayed on the screen in real time.
Users can view details such as temperature, humidity, and weather conditions.
